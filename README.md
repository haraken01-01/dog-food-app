# 犬の食事量・手作りトッピング支援 MVP

健康な成犬向けの一般的な食事量・トッピング支援ツールです。1日の推定カロリー、主食フード、手作り食材・トッピングのカロリーと主要栄養素、危険食材や偏りの注意を確認できます。

このアプリは診断、治療、療法食設計を行いません。完全手作り食の栄養バランスも保証しません。計算値は開始目安として扱い、体重、体型、便、食欲を見て調整してください。持病、療法食、子犬、妊娠・授乳期の場合は獣医師に相談してください。

## 実装範囲

- 犬情報入力フォーム
- 1日の推定カロリー計算
- 主食フード入力
- ドライフード、ウェット・フレッシュ系など複数の主食フード入力

- 食材ごとのカロリー・主要栄養素計算
- 危険食材チェック
- トッピング10%超え警告
- 肉中心・Ca:Pバランス注意
- 結果表示
- 参考資料ページ

## 入力上の考え方

- ココグルメなどのフレッシュフードは、パッケージ上で総合栄養食なら「総合栄養食ウェット・フレッシュ系」として入力します。
- 「総合栄養食ではない補助食（一般食・副食）」は、それだけで栄養が完結しない補助的なフードを指します。
- 手作り食材の用途で「トッピング」「おやつ」を選ぶと、10%目安の判定対象になります。
- 「主食の一部」を選ぶと総カロリーには含めますが、トッピング過多の判定からは外します。
- 避妊・去勢手術はチェック式です。未チェックの場合、カロリー計算上は手術を受けていない前提で扱います。
- 食材入力は1欄にまとめています。候補から選ぶと栄養計算でき、候補外の自由入力は危険食材チェックのみ行います。
- 調理方法は、同じ食材の調理法別データが登録されている場合は栄養値の切り替えに使います。未登録の場合は現在選ばれている候補食材の成分値を使います。
- 栄養メモに出る食品例は、追加量や治療目的の提案ではありません。個別の不足補正やサプリ量の提案は行いません。

## 初期データ

食材の栄養値は `src/data/food_nutrients.json` に置いています。MVPでは仮値を含むため、正式公開前に日本食品標準成分表などの正式データで差し替えてください。

危険食材は `src/data/dangerous_foods.json` に置いています。
